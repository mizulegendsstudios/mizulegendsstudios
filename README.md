# 🌀 Mizu Legends Studios
### *The Fractal Nexus for Web, Software & Interactive Labs*

Bienvenido al centro neurálgico de **Mizu Legends Studios**. Este portal, desplegado en `mizulegendsstudios.pages.dev`, utiliza el **Mizu-Fractal-Engine v2.3.0** para ofrecer una navegación de alta jerarquía, fluidez cinestésica y accesibilidad universal.


---

## 🏗️ Arquitectura del Ecosistema
El estudio se divide en cuatro nodos principales que reflejan nuestra versatilidad técnica y estratégica:

### 1. Mizu Research Nexus (Core Suite)
El laboratorio de herramientas fundamentales y recursos educativos.
* **Mizu Suite:** Aplicaciones de productividad (Chat, Maps, Math, News).
* **Mizu University:** Módulos de aprendizaje y transferencia de conocimiento.
* **Web & Game Parts:** Repositorio de componentes para desarrolladores.

### 2. Mizu Enterprise Suite (Efficiency Tools)
Software diseñado para la optimización de tareas y procesamiento de datos.
* **Mizu Speech App:** Suite de interpretación, conversión y OCR avanzado (`Mizu OCRx`).
* **Mizu Keygen & Task:** Herramientas de seguridad y gestión de procesos.
* **Mizu Coder:** El entorno de desarrollo interno para la suite.

### 3. Mizu Interactive Labs (Gaming & Simulation)
Donde la lógica de los eSports se encuentra con la simulación.
* **Moba Manager Simulator:** Nuestro proyecto insignia de simulación competitiva (In-Dev).
* **Fenix Chess:** Motor de ajedrez offline y variantes experimentales (`Spiders`).
* **Interactive Experiments:** Proyectos como `Ritmo Infinito` y `PixelversoBETA`.

### 4. Mizu Websites (Commercial Portfolio)
Consultoría y desarrollo a medida para sectores tradicionales y modernos.
* **Local Business:** Sastrería Los Abuelos, Elevadores Nacional.
* **Specialized:** Iglesia Adventista, Gatos Fundación, Furniture Store.

## 🛠️ Especificaciones Técnicas

* **Engine Core:** [Mizu-Fractal-Engine v2.3.0](https://github.com/tu-usuario/mizu-fractal-engine).
* **Frontend:** Vanilla JavaScript (ES6+), CSS3 con variables dinámicas, HTML5.
* **UX/UI:** Sistema de navegación por niveles (1, 3, 5, n), Kinetic Flow en eje Z y soporte nativo para Gamepads/Smart TVs.
